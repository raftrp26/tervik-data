# tervik-data

## Tervik Data Platform Architecture

This repository contains a personal project focused on designing and structuring a modern data platform architecture for Tervik’s payment ecosystem.

The project explores the end-to-end construction of a scalable analytics environment, covering the full data lifecycle from raw ingestion to business-ready data models (**OBT — One Big Table**).

---

## Scope

The repository includes studies and implementations related to:

- Raw data ingestion and staging layers
- Data warehouse modelling
- Transformation pipelines using **dbt**
- Intermediate and business layers
- OBT (One Big Table) construction for analytics and reporting
- Data quality and model organization practices
- Cloud-based analytics infrastructure concepts

---

## Main Technologies

- SQL
- dbt
- Google BigQuery
- Google Cloud Platform (GCP)
- Python

---

## Objective

The main goal of the project is to simulate and validate scalable data architecture patterns that could support analytics, growth, financial and operational use cases within a global payments and fintech environment.

The repository also serves as a learning and experimentation environment for:
- Analytics Engineering
- Data Modelling
- Modern Data Stack practices
- Data Transformation workflows
- Business-oriented data architecture
